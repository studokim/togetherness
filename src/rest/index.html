This server doesn't provide any HTML pages. For API, refer to /api.

Available API

  • GET /api/player/:name -> {player={name, avatar_id, faction_id}, error} — проверяем в реальном времени, что имя не занято; пока «занято», не разрешаем переходить на экран выбора фракции
  • POST /api/player {player} -> {ok, error, timer} — регистрируем игрока
  • POST /api/action {action={subject_name, object_name, action_id}} -> {ok, error, timer} — взаимодействуем, в qr-коде было зашифровано имя игрока-объекта
  • GET /api/action/:name -> {[action], error, timer} — показываем взаимодействия с участием игрока как в роли субъекта, так и объекта
  • GET /api/gold/:name -> {int, error} — показываем количество золота, надо обновлять каждые несколько секунд
