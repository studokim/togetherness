This server doesn't provide any HTML pages. For API, refer to /api.

Available API

  • POST /api/player {player={id, name, avatar_id, faction_id}} -> {ok, error, timer} — регистрируем игрока
  • GET /api/player/:id -> {player, error, timer} — получаем данные игрока-объекта при взаимодействии
  • POST /api/action {action={subject_id, object_id, action_id}} -> {ok, error, timer} — взаимодействуем, в qr-коде был зашифрован id игрока-объекта
  • GET /api/action/:id -> {[{action_id, as_subject, as_object}], error, timer} — показываем взаимодействия с участием игрока как в роли субъекта, так и объекта
  • GET /api/gold/:id -> {int, error} — показываем количество золота, надо обновлять каждые несколько секунд
